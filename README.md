# web-scraper
Python-based basic web scraper made to compare prices of certain services

## Status
This is a learning project. It successfully demonstrates:
- Web scraping with BeautifulSoup
- Following links programmatically  
- CSV data export

## Limitations
- Requires manual URL curation
- Works best on sites with simple HTML structure
- May need site-specific adjustments

## Future improvements
- Add database storage
- Improve price detection accuracy
- Handle JavaScript-rendered content
